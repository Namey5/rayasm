# rayasm
Simple raylib project written in x64 assembly.

https://github.com/user-attachments/assets/1afa0438-6ec5-425c-a57f-91a7adacb422

## building
Here's what I used in testing:

| Dependency | Version |
| ---------- | ------- |
| GNU Make   | 4.4.1   |
| GCC        | 15.1.1  |
| pkg-config | 2.5.1   |
| raylib     | 5.5     |

```bash
cd rayasm

make
./build/rayasm

# Alternatively:
make run
```
